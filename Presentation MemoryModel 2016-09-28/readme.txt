This is the presentation I gave av Sweden C++ meeting at Tritech.
THe presentation is about the memory model that was introduced in C++11.
The purpose is to give a high level overview of the old model, why
the old model is getting outdated on modern hardware and then
preset the most important changes made to C++11 to allow threads
within the language.
